## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Running with docker

Requires a computer with Docker installed.

```bash
# run command
$ docker compose up
```

## Running normal

Computer requirements need to install NodeJS, NestJS and mongodb.

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
